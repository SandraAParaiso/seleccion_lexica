# seleccion_lexica
mplementación web de una adaptación del experimento de selección léxica de Schmitt et al. (2001). Pone a prueba el modelo de acceso léxico de Levelt, Roelofs &amp; Meyer (1999) midiendo tiempos de respuesta en dos condiciones experimentales: conceptual y sintáctica. Ejecutable directamente en el navegador, sin instalación.
